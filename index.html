Site Informativo


O tema desse site é "Como é uma estrutura básica de um HTML?".

QUE SÃO TAGS:

Tags são marcações usadas em html, ou Linguagem de Marcação de Hipertexto, são o básico de toda estrutura em html.

TAGS NORMALMENTE UTILIZADAS:

'h1', 'h2', 'h3', 'h4', 'h5', 'h6' são utilizadas para titulo, quando maior o número atrás do "h" menor é a letra.

LISTAS:

Listas são feitas usando 'ul', 'ol', e 'li'.


EXEMPLO:

Lista não ordenada:
  • Item 1
  • Item 2

Lista ordenada:
  1. Item 1
  2. Item 2

Tag de Ancora(LINK):

Nossa 4° tag é a 'a', ela é um tag ancora.
